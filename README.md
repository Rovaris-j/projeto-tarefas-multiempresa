
 Projeto Tarefas Multiempresa

Sistema web de gerenciamento de tarefas com suporte a múltiplas empresas, desenvolvido com Laravel, Vue.js e Docker, seguindo boas práticas de organização, autenticação e isolamento de ambiente.


Tecnologias Utilizadas

Backend
- PHP 7.4
- Laravel 8
- MySQL 8
- Laravel Sanctum (autenticação)

###Frontend
- Vue.js
- Axios

Infraestrutura
- Docker
  
Antes de iniciar, certifique-se de ter instalado:
- Docker
- Docker Compose
- Git
